# 💳 Sistema Bancário Simples em Java

## 📌 Descrição
Este projeto consiste em um sistema bancário simples desenvolvido em Java, executado via terminal. O objetivo é simular operações básicas de uma conta corrente, permitindo interação do usuário através de um menu dinâmico.

O sistema foi desenvolvido como parte dos estudos da formação da Alura, com foco em lógica de programação, estruturas de controle e manipulação de dados.

---

## ⚙️ Funcionalidades
O sistema permite realizar as seguintes operações:

- ✔️ Consultar saldo  
- 💰 Receber valores (depósito)  
- 🔁 Transferir valores  
- 🚪 Sair do sistema  

---

## 🧠 Conceitos aplicados

Durante o desenvolvimento, foram utilizados conceitos fundamentais de Java:

- Estrutura de repetição (`do-while`)
- Estrutura condicional (`if`, `else if`, `switch`)
- Entrada de dados com `Scanner`
- Manipulação de variáveis
- Controle de fluxo
